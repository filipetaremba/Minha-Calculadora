# Calculadora Python Turbo 3000

Uma calculadora simples, divertida e poderosa feita em Python!

## Dependencias
- python3
- pip install customtkinter
- pip install os(Instale se necessario, ma ja e nativa no python)
- pip install platform( Instale se necessario, ma ja e nativa no python)

## Funcionalidades
- Soma, subtração, multiplicação e divisão
- Potência e raiz quadrada
- Tratamento de erros
- Interface colorida e limpa
- Mensagem secreta (digite 9)


## Como usar
1. Tenha Python instalado
2. Execute: `python app.py`
